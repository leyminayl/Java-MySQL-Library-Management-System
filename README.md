### **📚 Library Management System**
A functional desktop application developed as a **mini-project for my Java Programming course.** This system focuses on efficient library record-keeping and book tracking, built using **Java GUI (Swing)** for the interface and **MySQL** for data storage.

### **🚀 Key Features**

- **Full CRUD Capabilities:** Allows users to seamlessly add new books, update existing records, and remove entries from the system.
- **Real-time Dashboard:** Features a tabular dashboard that enables instant viewing of book titles, authors, and publication years.
- **Status Tracking:** Integrated a feature to monitor whether a specific book is "Available" or "Borrowed".
- **Persistent Storage:** Utilizes MySQL to ensure all library data is securely stored and easily retrievable.

### **🛠️ Tech Stack**

- **Language:** Java
- **GUI Framework:** Java Swing
- **Database:** MySQL
- **Tools:** MySQL Workbench, Visual Studio Code

### **💡 What I Learned**
Through this project, I strengthened my skills in:

- Implementing Java Swing for desktop user interfaces.
- Connecting a Java application to a relational database using MySQL.
- Managing data integrity through CRUD operations.

**📊 Main Dashboard & Inventory Management** This is the central hub of the system, designed for efficient data handling. It includes:

- **Real-time Tabular View:** A dynamic table that displays all books, including their ID, Title, Author, Year, and current Status (Available/Borrowed).
- **CRUD Control Panel:** Located on the left panel, this area contains input fields and action buttons for Adding new books and Updating existing records.
- **Data Management:** A dedicated Delete Record button is positioned for easy database maintenance.
- **Database Integration:** All entries shown in the dashboard are fetched in real-time and securely stored within a MySQL database.
